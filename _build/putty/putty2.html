Plink Commands¶

  1. This command will license the bigip: “C:Program FilesPuTTYplink.exe” -l admin -pw admin 10.1.1.7 run /util bash -c “/usr/local/bin/SOAPLicenseClient –basekey J1701-07024-25118-65122-2084674”

  2. This command will login to the bigip and change the password to what is in the master-key.txt file: “C:Program FilesPuTTYplink.exe” -l admin -pw admin 10.1.1.7 run /util bash -c “sh load-ucs.sh”

    load-ucs.sh

    cat master-key.txt |tmsh modify /sys crypto master-key prompt-for-password
tmsh save /sys config
tmsh load /sys ucs initial-config.ucs no-license
